Samples Page and Toolbar

Use the options on the Samples page to view and manage files relating to your files.

From the Samples tab, you can access the top toolbar icons described in Table 1.

Samples tab icons

Icon

Description

Create New Sample — allows you to upload samples files and create a matching record that appears in the grid

Search — allows you to search through file names and metadata to display a corresponding list of sample files

Download CSV — creates a list of all active sample files with metadata in a CSV file

Print — allows you to print the current view of your active samples

View Columns — allows you to show or hide various available columns

Filter Table — allows you to filter samples based on a single or multiple parameters

Edit — allows you to view the status of the pipeline

Remove — allows you to remove a sample record from the grid

Removed samples are archived rather than deleted.

Restore — allows you to restore an archived sample record to the active samples list

For information on archiving and restoring files, see Archiving and Restoring Sample Records on page 1.

From the bottom toolbar you can

Specify the number of rows per page (10, 15, or 100)

Use the < and > arrows to navigate to subsequent or previous pages
