Downloading a List of Sample Files

To download a list of active sample files to a .csv file

In the left panel, click Samples.

Click the Download icon in the toolbar.

The file automatically appears in your Downloads folder.

Double-click the file to open it and view the results.

The file contains information under the following headings:

Name

Owner

Created (date the sample file was created)

Pipeline (pipeline details)

Tags (tags associated with the sample file)

SampleID

Actions

DateasString (download date)
