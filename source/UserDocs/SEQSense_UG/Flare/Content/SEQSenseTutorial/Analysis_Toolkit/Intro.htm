Introduction

The is a Docker container with command line scripts and libraries that process FASTQ files as input for secondary analysis, and produces BAM files, count matrices, and reports as output for tertiary analysis.

This tutorial illustrates how the is used with the Complete Stranded RNA Library Prep Kit, and provides the necessary information, scripts, and libraries to analyze the Complete RNA data.

Instructions for obtaining human, rat, and mouse reference genomes for analysis are provided in Appendix 1, Downloading the Reference Genome.

This tutorial is presented in an Ubuntu Terminal interface, but you can use the commands in any environment that supports UNIX commands.
